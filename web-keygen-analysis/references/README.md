## 如何新增条目

收到新平台 / 新风控产品 / 新算法资料后：

1. 在本目录新增一份 `<short-name>-notes.md`，文件首部放一张 5 行摘要表（**站点/产品 / 关键参数 / 算法栈 / 反检测亮点 / 来源**），随后是逐字技术内容（按"抓包 → AST 解混淆 → 参数追源 → 算法识别 → 补环境/纯算 → 验证"分节）。
2. 在本 README 索引表追加一行。
3. 如果该资料引入了**全新的工作流环节**（例如 WASM 风控、Service Worker 拦截、WebRTC 指纹），再到 `../SKILL.md` 里**只补对应一节** Path，不要重写已有路径。
